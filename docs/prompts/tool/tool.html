使用指南

黑屏模式

黑屏模式通过提供纯黑色背景，最大限度减少视觉干扰，适合需要高度专注的编程、写作等任务。

白屏模式

白屏模式提供明亮的背景，适合阅读和内容创作，有助于减轻长时间工作的眼睛疲劳。

透明度调节

根据任务需求调整透明度：0-30%适合完全专注的深度工作，30-60%适合需要参考其他资料的工作。

定时器功能

基于番茄工作法的定时器，帮助您科学管理工作和休息时间，提高工作效率和持续性。
