SuperBlackScreen 是一款专业的专注工具，帮助您消除干扰，提高工作效率。只需几秒钟，即可创建完美的专注环境。

## 使用说明

1. 选择您喜欢的屏幕模式（黑屏、白屏、彩色或渐变）
2. 调整透明度至舒适水平
3. 设置专注时间和休息时间
4. 点击"全屏启动"开始您的专注会话
